# NOTTO - 이름별 로또번호 추천

이름을 기반으로 매주 로또번호를 추천해주는 사이트입니다.

## 현재 상태

더미 데이터 단계 - 등록된 이름 목록에서 추천번호를 확인할 수 있습니다.
추후 서버 연동 시 HTMX를 활용한 서버 통신으로 전환 예정입니다.

## 기능

- 현재 회차 정보 표시 (동행복권 API)
- 이름별 추천 로또번호 6개 리스트
- 이름 실시간 검색 필터링

## 기술 스택

- HTMX (CDN)
- PicoCSS (CDN)
- 순수 JavaScript

## 실행

`index.html`을 브라우저에서 직접 열면 됩니다.
